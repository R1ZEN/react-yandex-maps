# ListBox

Renders Yandex.Maps API [ListBox][control] control

## Basic Usage

<Canvas>
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <ListBox data={{ content: 'Select city' }}>
        <ListBoxItem data={{ content: 'Moscow' }} />
        <ListBoxItem
          data={{
            content: 'Saint Petersburg',
          }}
        />
      </ListBox>
    </Map>
  </YMaps>
</Canvas>
