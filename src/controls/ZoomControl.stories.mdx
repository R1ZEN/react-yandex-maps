# ZoomControl

Renders Yandex.Maps API [ZoomControl][control] control

## Basic Usage

<Canvas>
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <ZoomControl options={{ float: 'right' }} />
    </Map>
  </YMaps>
