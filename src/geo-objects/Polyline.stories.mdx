# Polyline

Renders Yandex.Maps API [Polyline][docs] object

## Basic Usage

<Canvas>
  <YMaps>
    <Map
      defaultState={{
        center: [55.72, 37.44],
        zoom: 10,
      }}
    >
      <Polyline
        geometry={[
          [55.8, 37.5],
          [55.8, 37.4],
          [55.7, 37.5],
          [55.7, 37.4],
        ]}
        options={{
          balloonCloseButton: false,
          strokeColor: '#000',
          strokeWidth: 4,
          strokeOpacity: 0.5,
        }}
      />
    </Map>
  </YMaps>
</Canvas>
