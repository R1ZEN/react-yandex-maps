# Circle

Renders Yandex.Maps API [Circle][docs] object

## Basic Usage

<Canvas>
  <YMaps>
    <Map defaultState={{ center: [55.76, 37.64], zoom: 10 }}>
      <Circle
        geometry={[[55.76, 37.6], 10000]}
        options={{
          draggable: true,
          fillColor: '#DB709377',
          strokeColor: '#990066',
          strokeOpacity: 0.8,
          strokeWidth: 5,
        }}
      />
    </Map>
  </YMaps>
</Canvas>
