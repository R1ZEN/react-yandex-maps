# FullscreenControl

Создает объект Yandex.Maps API [FullscreenControl][control]

## Использование

<Canvas>
  {`
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <FullscreenControl />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `data`: Control data
- `defaultData`: Uncontrolled control data
- `options`: Control options
- `defaultOptions`: Uncontrolled control options
- `state`: Control state
- `defaultState`: Uncontrolled control state

[control]:
  https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/control.FullscreenControl-docpage/
