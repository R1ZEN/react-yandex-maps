# GeolocationControl

Renders Yandex.Maps API [GeolocationControl][control] control

## Basic Usage

<Canvas>
  {`
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <GeolocationControl options={{ float: 'left' }} />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `data`: Control data
- `defaultData`: Uncontrolled control data
- `options`: Control options
- `defaultOptions`: Uncontrolled control options
- `state`: Control state
- `defaultState`: Uncontrolled control state

[control]: https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/control.GeolocationControl-docpage/
