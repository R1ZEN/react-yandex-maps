# GeoObject

Renders Yandex.Maps API [GeoObject][docs] object

## Basic Usage

<Canvas>
  {`
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 3,
      }}
    >
      <GeoObject
        geometry={{
          type: 'LineString',
          coordinates: [
            [55.76, 37.64],
            [52.51, 13.38],
          ],
        }}
        options={{
          geodesic: true,
          strokeWidth: 5,
          strokeColor: '#F008',
        }}
      />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `geometry`: Circle geometry
- `defaultGeometry`: Uncontrolled circle geometry
- `properties`: Circle properties
- `defaultProperties`: Uncontrolled circle properties
- `options`: Circle options
- `defaultOptions`: Uncontrolled circle options
- `dangerZone`: Used in a special case where constructor needs to be modified before passing it to the mount method

[docs]: https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/GeoObject-docpage/
