# Circle

Renders Yandex.Maps API [Circle][docs] object

## Basic Usage

<Canvas>
  {`
  <YMaps>
    <Map defaultState={{ center: [55.76, 37.64], zoom: 10 }}>
      <Circle
        geometry={[[55.76, 37.6], 10000]}
        options={{
          draggable: true,
          fillColor: '#DB709377',
          strokeColor: '#990066',
          strokeOpacity: 0.8,
          strokeWidth: 5,
        }}
      />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `geometry`: Circle geometry
- `defaultGeometry`: Uncontrolled circle geometry
- `properties`: Circle properties
- `defaultProperties`: Uncontrolled circle properties
- `options`: Circle options
- `defaultOptions`: Uncontrolled circle options
- `dangerZone`: Used in a special case where constructor needs to be modified
  before passing it to the mount method
- `instanceRef`: Object instance

[docs]: https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/Circle-docpage/
