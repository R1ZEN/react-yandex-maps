# TypeSelector

Renders Yandex.Maps API [TypeSelector][control] control

## Basic Usage

<Canvas>
  {`
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <TypeSelector options={{ float: 'right' }} />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `mapTypes`: Array of constructors for map types or keys. If the parameter is
  omitted, the item is added to the standard set of map types. (`yandex#map`,
  `yandex#satellite`, `yandex#hybrid`)
- `defaultMapTypes`: Uncontroled array of constructors for map types or keys. If
  the parameter is omitted, the item is added to the standard set of map types.
  (`yandex#map`, `yandex#satellite`, `yandex#hybrid`)
- `options`: Control options
- `defaultOptions`: Uncontrolled control options
- `state`: Control state
- `defaultState`: Uncontrolled control state

[control]:
  https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/control.TypeSelector-docpage/
