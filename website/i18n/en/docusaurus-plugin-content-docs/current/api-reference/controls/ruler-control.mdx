# RulerControl

Renders Yandex.Maps API [RulerControl][control] control

## Basic Usage

<Canvas>
  {`
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],
        zoom: 9,
        controls: [],
      }}
    >
      <RulerControl options={{ float: 'right' }} />
    </Map>
  </YMaps>
`}
</Canvas>

## Props

- `data`: Control data
- `defaultData`: Uncontrolled control data
- `options`: Control options
- `defaultOptions`: Uncontrolled control options
- `state`: Control state
- `defaultState`: Uncontrolled control state
- `instanceRef`: Object instance

[control]:
  https://tech.yandex.com/maps/doc/jsapi/2.1/ref/reference/control.RulerControl-docpage/
